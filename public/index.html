How to use:

Input inventory in the following format:

Blackrock
Blackrock
Tiamat

Legend:

You have or can make this card.

You're missing a card to make this.

You don't have this card.

Long term goal

To build a tool to help you figure out what you can fuse, what cards you need to build an awesome card. And to eventually help you decide which card to make, with the help of tu_optimize.
